
• settings.json: "typescript": "npx ts-node --files" → permite executar o ts pelo code-runner.

• ts-node --help → para configurar a compilação se necessário.

• npx tsc [--init] [-w] → compila TS gerando output JS → [--init] início de projeto padrão → [-w] opcional ficará assistindo modificações.
! Em produção, normalmente se usa o bundler (webpack) para se fazer o build.

• Em caso de migração de códigos, desative o "strict" em tsconfig.
